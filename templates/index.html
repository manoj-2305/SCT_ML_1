    About Smart House Price Predictor

    Overview: This internship project focuses on building a predictive web application using machine learning. The model helps users estimate housing prices based on property features like square footage, bedrooms, and bathrooms.

    Dataset: We used the Kaggle Ames Housing dataset containing a detailed record of real estate sales. Only the most relevant features were used for this prototype to keep the model simple and explainable.

    Modeling Approach: Linear Regression was chosen for its simplicity and interpretability. It maps features to prices using coefficients. The model was evaluated using metrics like Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and RÂ² Score (0.6579).

    Deployment: The model is deployed in a Flask web app. Users input their property details in the form, and the server responds with predicted house prices. Real-time conversion from USD to INR and formatted output is included.

    Frontend Features: The application has a responsive layout with a left-side input form and right-side dynamic prediction history that clears on reload. History is stored in localStorage until refresh, enhancing UX without permanent storage.

    Tech Stack: Python, Pandas, Scikit-learn, Flask, HTML5, CSS3, JavaScript ES6. The UI has been improved with modern styling and responsive layout techniques.

    Pros:

    • Simple and accurate for the basic use-case
    • Good UX with responsive design
    • Modular design (easy to add models or features)
    • Real-time prediction and temporary history

    Limitations:

    • Does not account for location-specific data or market fluctuations
    • History is not stored permanently
    • Limited feature set

    Future Scope: Add location data, connect to a real estate API, add user accounts and save permanent history, integrate advanced ML algorithms (Random Forest, XGBoost), or even deploy on cloud (Heroku, AWS, or Streamlit Cloud).
